入學前準備階段：為成功打好基礎

我們深信，成功的學習始於個人化的起點。因此，我們設計了獨特的入學前評測與彈性補強課程，確保每位學員都能在最適合自己的跑道上起跑。

能力評測與分流機制 (2小時)

  • 程式設計基礎測試：評估Python或其他語言經驗
  • 電機電子基礎測試：電路、控制系統基礎知識
  • 數學統計能力測試：微積分、統計學、線性代數基礎
  • 工業製程認知測試：製造業經驗與工業常識

基礎補強課程 (彈性10-30小時)

根據評測結果，學員可選擇參加以下最適合自己的補強模組：

電機電子基礎 (15小時): 專為非電機背景學員設計
數學統計補強 (10小時): 專為數學基礎薄弱者設計
Python程式入門 (15小時): 專為無程式經驗者設計

完整課程規劃 (260小時核心 + 30小時彈性)

第1章: 鋼鐵業智慧製造導論 (10小時)

學習目標：建立鋼鐵業整體認知與智慧製造概念。

子章節時數學習內容
1.1 鋼鐵製造流程4小時從煉鐵到軋鋼的完整生產流程
1.2 工業4.0與智慧製造4小時在鋼鐵業的應用藍圖與挑戰
1.3 儀電工程師角色轉變2小時傳統儀電 vs 智慧製造儀電工程師

實習安排：鋼鐵廠參訪。

第2章: 儀電工程實務技能 (40小時)

學習目標：建立完整的儀電工程師實務操作能力。

2.1 PLC程式設計實務 (15小時)
時數分配學習內容實習設備
8小時梯形圖程式開發與除錯技術西門子S7-1200實習台×6
4小時西門子/三菱PLC系統操作實務TIA Portal軟體
3小時HMI人機介面設計與整合WinCC軟體
2.2 馬達驅動系統實務 (12小時)
時數分配學習內容實習設備
6小時變頻器參數設定與調試技術ABB/西門子變頻器×6台
4小時伺服馬達控制實務操作伺服馬達實習台
2小時驅動系統故障診斷與排除故障模擬設備
2.3 工業感測器與儀表實務 (8小時)
時數分配學習內容實習設備
4小時溫度/壓力/流量感測器校準各式感測器套件
4小時數據採集系統建置與測試DAQ系統、LabVIEW
2.4 電力系統基礎實務 (5小時)
時數分配學習內容實習設備
3小時高低壓配電系統維護實務電力系統模擬器
2小時電力品質分析與測試電力品質分析儀

第3章: AI程式設計與大數據分析實務 (40小時)

  • 3.1 Python程式設計進階 (15小時): 資料結構、物件導向、Pandas/Numpy應用。
  • 3.2 工業大數據分析 (15小時): 從PLC/SCADA系統獲取數據、統計分析與特徵工程。
  • 3.3 生成式AI提示工程 (10小時): GPT/Gemini應用、程式開發輔助、倫理與風險管理。

第4章: 人工智慧與機器學習 (30小時)

  • 4.1 監督式學習演算法 (15小時): 學習迴歸分析、分類演算法、模型評估。
  • 4.2 非監督式學習與異常檢測 (10小時): 學習聚類分析、Isolation Forest等。
  • 4.3 模型部署與MLOps (5小時): 學習模型序列化、監控與更新機制。

第5章: 工業時間序列分析與預測 (25小時)

  • 5.1 時間序列基礎 (10小時): 特性分析、數據預處理。
  • 5.2 傳統時間序列模型 (8小時): ARIMA模型、指數平滑法。
  • 5.3 深度學習時間序列預測 (7小時): LSTM神經網路應用與案例演練。

第6章: AI影像辨識原理與實務 (20小時)

  • 6.1 電腦視覺基礎 (8小時): OpenCV操作、SIFT/HOG特徵提取。
  • 6.2 深度學習影像辨識 (8小時): CNN原理、遷移學習應用。
  • 6.3 工業影像應用開發 (4小時): 缺陷檢測系統、邊緣運算與影像串流處理。

第7章: 專案一 - 產線預知保養(PdM) (25小時)

  • 7.1 專案需求分析與系統設計 (5小時)
  • 7.2 數據採集與預處理 (8小時)
  • 7.3 預測模型開發 (8小時)
  • 7.4 系統整合與測試 (4小時)

第8章: 專案二 - AI工安管理系統 (25小時)

  • 8.1 工安AI系統需求與設計 (5小時)
  • 8.2 火警監測AI開發 (10小時)
  • 8.3 人員安全監控系統 (10小時)

第9章: 專案三 - 智慧能源管理系統 (20小時)

  • 9.1 能源管理系統規劃 (4小時)
  • 9.2 能耗預測模型開發 (10小時)
  • 9.3 能耗異常檢測與優化建議 (6小時)

第10章: 生成式AI產業應用與自動化 (5小時)

  • 10.1 專案報告自動生成 (2小時): RAG技術應用
  • 10.2 RPA流程自動化 (3小時): 維護/安全報告自動化

第11章: 專題製作實務與成果發表 (20小時)

  • 11.1 整合專案規劃 (4小時): 跨系統整合設計、敏捷開發方法
  • 11.2 系統開發與測試 (12小時): 模組數據串接、功能/效能測試
  • 11.3 成果發表與評量 (4小時): 向業界專家展示專案成果

認證考試與就業輔導

進行包含筆試(40%)、實作(40%)與專題(20%)的結訓認證，並提供技術履歷撰寫、GitHub作品集整理、模擬面試與合作企業媒合服務。

課程與培訓需求對應

1. 產線預知保養、維修

核心專案：第三模組 / 專案一：產線預知保養(PdM)
基礎知識：第一模組(工業基礎)、第二模組(機器學習與時間序列分析)

說明：透過完整的專案實作，學員將親手處理設備的振動、電流數據，建立異常偵測與故障預測模型，實現從「事後維修」到「預知保養」的思維與技能轉變。

2. 工安管理 (例如火警監測)

核心專案：第三模組 / 專案二：AI工安管理系統
核心技術：第二模組 / AI影像辨識原理與實務

說明：專門設立影像辨識的理論與實作章節，並在專案中讓學員實際開發「火焰煙霧偵測」、「安全帽佩戴識別」等工安應用，完全符合需求。

3. 能源管理 (能耗異常偵測、預測)

核心專案：第三模組 / 專案三：智慧能源管理系統
核心技術：第二模組 / 工業時間序列分析與預測

說明：學員將學習建立工廠用電量的預測模型，並利用機器學習找出異常耗能的設備或時段，提供具體的節能改善依據與AI優化建議。

4. 儀電工程師專業能力培訓

整體課程架構，特別是：第一模組 / 儀電工程實務技能

說明：課程的基石在於賦能(Empower)傳統儀電工程師。第一模組的PLC、感測器、馬達、電力系統等內容，正是儀電工程師日常工作的數據來源，本課程教導他們如何利用AI分析這些數據，全面升級其專業能力。
